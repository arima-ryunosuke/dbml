  • public

    bindValue ($param, $value, $type = Doctrine\DBAL\ParameterType::STRING) : bool

    Binds a parameter value to the statement.

    The value can optionally be bound with a DBAL mapping type.
    If bound with a DBAL mapping type, the binding type is derived from the mapping
    type and the value undergoes the conversion routines of the mapping type before
    being bound.

    Parameters
    stringint $param

    The name or position of the parameter.
    mixed $value

    The value of the parameter.
    mixed $type = Doctrine\DBAL\ParameterType::STRING

    Either a PDO binding type or a DBAL mapping type name or instance.
    Return
    bool

    TRUE on success, FALSE on failure.
    Throws
    Doctrine\DBAL\Exception
