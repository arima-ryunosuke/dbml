class Doctrine\Common\Cache\CacheProvider

Base class for cache provider implementations.

Construction

Implements
Doctrine\Common\Cache\Cache Interface for cache drivers.
Doctrine\Common\Cache\FlushableCache Interface for cache that can be flushed.
Doctrine\Common\Cache\ClearableCache Interface for cache that can be flushed.
Doctrine\Common\Cache\MultiOperationCache Interface for cache drivers that supports multiple items manipulation.
Doctrine\Common\Cache\MultiPutCache Interface for cache drivers that allows to put many items at once.
Doctrine\Common\Cache\MultiDeleteCache Interface for cache drivers that allows to put many items at once.
Doctrine\Common\Cache\MultiGetCache Interface for cache drivers that allows to get many items at once.

    getStats ()

    Retrieves cached information from the data store.

    The server's statistics array has the following values:

    • hits
      Number of keys that have been requested and found present.

    • misses
      Number of items that have been requested and not found.

    • uptime
      Time that the server is running.

    • memory_usage
      Memory used by this server to store items.

    • memory_available
      Memory allowed to use for storage.

    Parameters
    Return
    arraynull

    An associative array with server's statistics if available, NULL otherwise.
